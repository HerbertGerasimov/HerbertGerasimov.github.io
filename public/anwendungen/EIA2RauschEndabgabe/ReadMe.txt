Anwendung: [Cine Dventure]
Wichtige Hinweise zur Ausführung:
Projektordner importieren:

Importieren Sie den gesamten Anwendungsordner in Visual Studio Code (VS Code) über "File" -> "Add Folder to Workspace".
Live Server-Erweiterung installieren:

Überprüfen Sie, ob die "Live Server"-Erweiterung in VS Code installiert ist. Falls nicht, finden Sie diese im Extensions Marketplace.
Index.html mit Live Server öffnen:

Klicken Sie mit der rechten Maustaste auf die "index.html"-Datei im Projektverzeichnis und wählen Sie "Open with Live Server".
Anwendung genießen:

Ihr Standard-Webbrowser öffnet sich, und die Anwendung wird auf einem lokalen Server gestartet. Jetzt können Sie die Anwendung problemlos verwenden und testen.

Wichtiger Hinweis:
Die Anwendung ist optimiert für die Ausführung auf einem lokalen Server wie "Live Server" in Visual Studio Code. Bitte folgen Sie diesen Schritten, um sicherzustellen, dass alle Funktionen korrekt funktionieren.
